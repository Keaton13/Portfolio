CSS Files and Structure

This theme has been used 7 CSS styles:

  • Bootstrap - layout and responsives
  • Et-line-icons - some of the icons used in the theme
  • FontAwesome-icons - some of the icons used in the theme
  • Owl carousel - styles for slider in testimonials and portfolio section in index2.html
  • Swiper - styles for slider in portfolio section in index.html
  • Animate - CSS animation
  • Styles - All visual styles

JavaScript

All plugins are located before end of the body tag.

    <!-- jQuery (necessary for Bootstrap's JavaScript plugins) -->
    <script src="js/jquery-3.2.1.min.js"></script>
    <script src="js/bootstrap.min.js"></script>
    <!-- owl-carousel plugin -->
    <script src="js/owl.carousel.min.js"></script>
    <!-- wow animation plugin -->
    <script src="js/wow.min.js"></script>
    <!-- validation plugin -->
    <script src="js/validator.min.js"></script>
    <!-- contact form js -->
    <script src="js/contact.js"></script>
    <!-- swiper js -->
    <script src="js/swiper.min.js"></script>
    <!-- main script, settings for all scripts -->
    <script src="js/scripts.js"></script>

Contact form settings

Make sure your host is support php files.

You can open contact.php in php folder ,contact.js in js folder for setting contact section. There is comment in files for explanation.
